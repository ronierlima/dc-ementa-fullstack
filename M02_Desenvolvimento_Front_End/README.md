# **M02 - Desenvolvimento Front-End Avançado**

## **Apresentação do Módulo**

O módulo de **Desenvolvimento Front-End Avançado** tem como objetivo capacitar os alunos a criar aplicações web dinâmicas, interativas e de alta qualidade utilizando **JavaScript**, **React** e **Cypress** para testes automatizados. Os participantes aprofundarão seus conhecimentos em manipulação do DOM com JavaScript, desenvolvimento de interfaces modernas e responsivas com React, e garantia da qualidade das aplicações por meio de testes automatizados com Cypress.

O aprendizado é construído de forma progressiva, começando pela manipulação avançada do DOM, avançando para o desenvolvimento com React e culminando na implementação de testes automatizados. Com isso, os alunos desenvolvem habilidades técnicas essenciais para o mercado, além de boas práticas de organização, desenvolvimento e garantia de qualidade.

---

## **Principais Objetivos do Módulo**

- **Aprofundar o Uso do JavaScript no Front-End:**  
  Capacitar os alunos a manipular o DOM de forma avançada, criando funcionalidades interativas e dinâmicas nas páginas web.

- **Desenvolver Aplicações Dinâmicas com React:**  
  Ensinar a construir interfaces modernas e escaláveis utilizando a biblioteca React, abordando componentes funcionais, Hooks, Context API e roteamento.

- **Garantir a Qualidade das Aplicações com Testes Automatizados:**  
  Introduzir práticas de testes automatizados utilizando Cypress, assegurando a confiabilidade e a robustez das aplicações desenvolvidas.

- **Integrar Conhecimentos para Desenvolvimento Web Completo:**  
  Combinar JavaScript, React e Cypress para criar aplicações web completas, eficientes e de alta qualidade, alinhadas com as demandas do mercado de trabalho moderno.

- **Preparar para Tecnologias Modernas e Boas Práticas:**  
  Estabelecer uma base sólida que permitirá aos alunos avançar para módulos mais complexos e adotar boas práticas de desenvolvimento e testes.

---

## **Estrutura do Módulo**

O módulo está organizado em **3 unidades**, totalizando **32 horas**, com aulas que combinam teoria e prática:

1. **JavaScript e DOM (20 horas):**  
   Manipulação avançada do DOM, eventos, consumo de APIs e gerenciamento de estado com `localStorage`, culminando em um projeto prático que integra todos os conceitos aprendidos.

2. **React (32 horas):**  
   Desenvolvimento de aplicações web dinâmicas e interativas utilizando React, abordando componentes funcionais, props, estado com Hooks e Context API, roteamento com React Router e estilização com PrimeReact.

3. **Testes de Frontend com Cypress (8 horas):**  
   Implementação e automatização de testes End-to-End utilizando Cypress, garantindo a qualidade e a confiabilidade das aplicações React desenvolvidas.

---

## **Destaques do Módulo**

- **Aprofundamento em JavaScript e Manipulação do DOM:**  
  Capacita os alunos a criar funcionalidades interativas e dinâmicas, manipulando elementos do DOM de forma avançada e consumindo dados de APIs externas.

- **Desenvolvimento com React:**  
  Foco no uso de componentes funcionais e Hooks para gerenciamento de estado e efeitos colaterais, alinhado com as melhores práticas atuais do React. Inclui a integração eficiente de bibliotecas de UI como PrimeReact para criar interfaces modernas e responsivas.

- **Garantia de Qualidade com Cypress:**  
  Introdução abrangente aos conceitos e importância dos testes automatizados no desenvolvimento frontend, utilizando Cypress para escrever, executar e manter testes End-to-End eficazes.

- **Projeto Prático Integrador:**  
  Desenvolvimento de um **Sistema de Cardápio** similar ao iFood, permitindo aos alunos aplicar todos os conceitos aprendidos em um cenário realista. O projeto envolve exibição de produtos categorizados, personalização de itens, gerenciamento de carrinho e integração com uma API pré-existente.

- **Boas Práticas de Desenvolvimento e Testes:**  
  Enfoque na organização eficiente do código, reutilização de componentes, manutenção da qualidade do código e otimização de testes, preparando os alunos para práticas profissionais no mercado de trabalho.

- **Integração de Ferramentas Modernas:**  
  Uso de ferramentas e bibliotecas populares no mercado, como React Router, PrimeReact e Cypress, preparando os alunos para ambientes de desenvolvimento profissionais e exigentes.
